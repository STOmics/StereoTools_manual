StereoMap 4.2.0 更新亮点

  • Image Processing图像处理：
    • Step1：上传图像时，若为QC拼接大图，无需填写显微镜参数。
    • Step2：形态学配准时，新增矩阵透明度调节功能。
    • Step4：细胞分割新增自动模型分割选项。
  • Tools：
    • TIF大图QC不再需要填写显微镜参数。
    • 在DAPI + mIF QC场景下，“校准评估”结果不再作为QC通过与否的决定因素。
    • 删除多通道荧光图像QC的适配。

系统要求

最低配置
Windows
  • Windows 10或更高的版本（64 位）
  • 内存：16 GB(仅小工具-QC支持最小8GB)
  • 强烈建议使用 SSD 固态硬盘
  • 建议缩放设置为 100%
MacOS
  • 仅适配了 M 系列的芯片
  • MacOS 11.3 (Big Sur) 或更高的版本
  • 内存：16 GB
  • 强烈建议使用 SSD 固态硬盘
  • 建议使用默认的设置

性能说明

  • 如果您要处理超大规模数据（如超过 10 万个细胞或 bin 数据），我们建议使用 32GB 及以上内存和四核处理器，以确保软件顺畅运行。
  • 如果您使用外部存储设备（比如移动硬盘）时，由于读写速度较慢或可能意外断开，运行速度可能变慢，甚至导致软件崩溃。

软件安装

下载.exe文件，双击可执行文件开始安装。

如果在安装的过程中遇到如下提示，您可以点击“打开”，将其添加到信任中。

选择安装路径，默认为"D:\StereoMap"，可根据需要修改目标文件夹。点击 Install 开始安装。

首次安装或者在线更新，请确保您的安装目录必须满足至少 20GB 的可用磁盘空间。

请确保安装路径中未包含其它特殊字符或者符号。当前支持的字符包括：空格、下划线、字母A~Z和数字0~9。

安装完成后，点击 Finish 即可。

相关软件（SAW）兼容

  • SAW >= v8.0 （历史版本的适配兼容信息）。
  • 请使用最新的 Image QC 重新QC，生成 TAR(.tar.gz)图像文件。

设置

在线更新

点击右上角的设置，选择 About ，联网状态下可以手动检查版本信息，如果有新版本发布，您可以选择下载并安装新的软件包。

您也可以勾选软件开机时自动检查最新版本的选项，程序会自动检测是否有新版本的更新。

保存

设置每个模块输出文件的路径，默认是您的安装目录，您也可以根据需求更改。
