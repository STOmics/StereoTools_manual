H&E图像

为什么要做H&E染色图像?

苏木精-伊红（Hematoxylin and eosin，H&E）染色是一种常用的组织染色，可为医学诊断提供组织学信息，被认为是金标准。苏木精主要将细胞核染成蓝紫色，而伊红主要将细胞质和细胞外基质染成不同深浅的粉红色，通过 Stereo-seq 生成的空间表达矩阵与 H&E 染色图像相结合，可以将细胞的形态与空间局部特征表达关联起来，挖掘组织切片中更多的信息。

H&E染色图像的注意事项

因 Stereo-seq 芯片是不透明的硅片，H&E 染色的图像需要用落射明场的显微镜成像，拍摄的背景颜色和轨迹线的颜色都接近为白色。与荧光灰度图像不同，H&E 染色图像通过 R-G-B 颜色的组合可以提供复杂的组织病理学信息，图像的文件大小比相同尺寸的灰度图像要大得多，故所需的计算资源也会相对较高。目前，StereoMap 在 16GB 的内存下，仅能够处理 24 位彩色、10 倍镜下1cm x 1cm 芯片的图像数据。

StereoMap 和 SAW 中处理的H&E染色图像为单张 24 位深的彩色图像。（有关更多信息，请参见 图像类型和格式 获取更多信息。）

彩色图像 24位的单张彩色图像

第一步：上传图像

您可以点击拖拽区域内的 Choose file 按钮，从本地选择一个兼容格式的图像文件，或者直接将文件拖拽到该区域中。此类型图像只支持上传一个文件。

请参考图像处理输入文件获取更详细的图像文件信息。

文件选择完成后，系统将自动解析图像并提取相关数据。解析时间会根据文件类型和大小有所差异。\ 在图像处理流程中，Stereo-seq 芯片的序列号（SN）信息非常关键。若您上传的是是.tar.gz 或者 .stereo 格式的文件，系统会直接从中读取相关信息并立即开始解析。

如果您上传的是 .tif 或 .tiff 格式的图像文件，请根据当前页面提示手动填写所需信息，并点击 Confirm 按钮后开始图像解析和质控分析。\ 待图像解析与质控分析完成后，点击 Next 进入第二步操作。

第二步：图像配准

在此步骤中，您需要调整图像的方向、角度和尺度，使其与空间特征表达矩阵对齐。在 Stereo-seq 数据分析中，StereoMap 提供了两种图像配准方法：

  1. 形态学配准：根据组织形态的相似度将图像和矩阵对齐。
  2. 打点配准：通过标记特定的 track 点，将图像对齐至由 Stereo-seq 芯片序列号（SN）推导出的轨迹线模板。该方法无需 SAW 生成的空间特征表达矩阵，但应考虑一些先决条件。

形态学配准

如果您在第 一步中：上传了 .stereo文件，进入第二步时，您就可以同时看到图像和空间特征表达矩阵。对于 .tar.gz 或 .tif/.tiff 格式的图像文件，在第二步您需要额外上传一个.stereo 文件来指定空间特征表达矩阵，点击 选择 .stereo 文件。

如果您的图像已经包含在.stereo 文件中，但您希望更改空间特征表达矩阵文件，您可以点击并加载一个新的.stereo 文件。这将重新加载矩阵文件，但依然使用第一步中的图像数据。

手动配准过程包括两个阶段：

粗略配准：根据形态学特征调整图像的方向

  • 使用翻转工具可以将图像绕Y轴镜像翻转。
  • 点击旋转按钮使用可以将图像以相同的方向旋转。

精细配准：调整图像位置和缩放，确保图像与空间特征表达图精确的重叠。

  • 使用平移面板 设置移动的步长和移动的四个方向（上、下、左、右）。
  • 使用缩放工具来调整图像的比例，使显微镜图像与空间特征表达图的尺寸一致。
  • 点击 Chip trackline 显示矩阵的参考轨迹线模板，辅助进行精准对齐。
  • 如果图像中的轨迹线较暗，可以手动调整图像的归一化、对比度、亮度和不透明度等工具按钮。

调整图像以便可以清楚的展示轨迹线。

完成配准的图像将标记为已完成。

如果做了手动配准，可以跳过步骤 3 和 4，您可以导出手动后的tar.gz图像文件，输入到 SAW 运行自动的组织分割和细胞分割。

另外，我们会输出*regist.tif 图像文件，该文件是配准后的图像，其形状和方向跟基因表达矩阵相匹配，此图像可以作为组织分割或者细胞分割的起点。如果您考虑使用第三方分割工具，强烈建议使用此图像文件作为输入，避免图像与 Mask 之间存在位移、角度和尺度的偏差。

打点配准

打点配准可以在不使用特征表达矩阵的前提下，利用图像上的 track 点以及预定义规则的芯片 mask 信息将图像对齐。

您需要确认图像以选择到正确的用于计算的特征点，可能会导致您的数据配准失败。如果无法达到上述要求，强烈建议您使用形态学配准的方式做后续分析。

使用该方法的前提条件：

  1. 图像 QC 通过，因为要确保 track 线是可见的。
  2. 有预定义规则的 Stereo-seq 芯片: Stereo-seq N FFPE V1.0, Stereo-seq T FF V1.3, or Stereo-CITE T V1.1 chips。
  3. 输入 .tar.gz 或 .stereo 文件中对应的 .tar.gz 是由 StereoMap >= 4.1 生成的。

为确保图像可以配准成功，需要您选择一个指定的点确定图像的方向，否则图像配准可能会失败。

选择特定的 track 点保证获取正确配准结果的要求如下：

  1. 图像中组织的方向应与其在载玻片上的实际放置方向一致，且载玻片上的刻字标签应位于右侧。允许的最大倾斜角度不得超过 15°。
  1. 确保图像的四个边/角可以被清晰的看到。
  2. 请确保按照最新的 Stereo-seq 空间多组学技术成像指南-3.2.2章节拍摄您的图像，可以从 STOmics - 文档库中访问该指南。

如果您的图像数据无法满足以上所有的要求，强烈推荐您使用形态学配准。

打点配准展示的图像数据为经过小角度旋转或者小尺度缩放的图像。

打点配准分为如下两个步骤：首先调整图像，确认芯片的四个边/角可以被清楚的看到。最后从已有的track点中选择某个track点将其作为参考点。

为保证图像的四个边/角可以被清晰的看到，您可以使用归一化、对比度、亮度和不透明度等工具调整图像。

看到芯片的四个角后，您需要确认靠近芯片左下角的第一个预设的 track 点，单击该点选中后，点击 Next 按钮即可完成配准。

完成打点配准后，系统会自动调整图像的方向，您可以在 Step 3 组织分割查看配准的结果。

如果做了手动配准，可以跳过步骤 3 和 4，您可以导出手动后的tar.gz图像文件，输入到 SAW 运行自动的组织分割和细胞分割。

另外，我们会输出*regist.tif 图像文件，该文件是配准后的图像，其形状和方向跟基因表达矩阵相匹配，此图像可以作为组织分割或者细胞分割的起点。如果您考虑使用第三方分割工具，强烈建议使用此图像文件作为输入，避免图像与 Mask 之间存在位移、角度和尺度的偏差，导致第三方分割的Mask文件无法导入。

第三步：组织分割

组织分割是个可跳过的步骤。

在该步骤中，您需要圈定组织区域，准确的识别组织边界可以有效减少背景干扰，提升聚类分析的准确性。系统会将图像中分割的组织结果映射到测序生成的空间特征表达矩阵上，形成组织区域的表达热图。

如果您在第一步中上传了.stereo文件，您可以在配准后的图像图层上看到一个半透明的组织掩膜。

如果是.tar.gz或.tif/.tiff图像文件，您需要使用手动工具来绘制组织区域。

在该步骤中，您可可以修改文件中记录的组织掩膜或新建一个新的掩膜。.tar.gz或.stereo文件中记录的组织掩膜将会在 Segmentation mask 下拉菜单中标记为 RECORD ，而您手动绘制或导入的掩膜将标记为 CUSTOM 。从 Segmentation mask 下拉菜单中选择所需的掩膜，即可在画布中查看并进行编辑。

在编辑组织区域的掩膜图像时，您可以灵活搭配使用套索、画笔和橡皮擦等工具。

  • 套索工具：适用于大面积区域的圈选，可自由绘制范围来选择或取消选择。
  • 画笔工具：适合细节处理，比如边缘修补或填补组织中的小孔洞。通过涂抹的方式操作更灵活。
  • 橡皮擦工具：与画笔类似，主要用于局部删除或精细地取消选择已圈定的区域。

如果你已经使用第三方分割工具创建了 .tif 格式二值化掩膜文件，可通过以下方式导入：

  • 在右侧面板点击 Segmentation mask下拉菜单
  • 点击 Add mask 来导入掩膜文件
  • 选择本地的 .tif 掩膜文件后点击 Open 打开，即可导入

如果对导入后的结果不满意，还可以替换掩膜：

  • 再次打开 Segmentation mask下拉菜单
  • 在 CUSTOM 分类中，选择已有掩膜后点击
  • 选择新文件并点击 Open 完成替换

第四步：细胞分割

细胞分割是个可跳过的步骤。

细胞分割是生成单细胞空间分辨率数据的重要环节，H&E 染色的图像，可以对核区域或者细胞区域做分割。

如果您在第一步中上传了.stereo文件，您可以在配准后的图像图层上看到一个红色的细胞/细胞核掩膜。

对于.tar.gz或.tif/.tiff图像文件，您需要使用手动工具来绘制细胞。

因只在组织区域的范围内展示对应的细胞分割结果，组织外的区域将会展示为白色的背景。

类似于组织分割，您可以选择编辑之前已经记录的掩膜（标记为 RECORD ）或创建一个新的掩膜（标记为 CUSTOM ）。通过从 Segmentation mask 下拉菜单中切换画布中展示的掩膜。

在 StereoMap 中，您可以通过以下三种方式优化细胞分割效果：

  1. 导入细胞分割掩膜\ 如果您已经用第三方工具对配准后的数据进行细胞分割，可以直接将结果导入 StereoMap，用于 Stereo-seq 的分析。如果您拥有高质量的分割结果并希望无缝整合至 Stereo-seq 工作流程，此方法最为适用。适合大规模数据或需要使用高级模型训练的情况。
  2. 参数可调的半自动工具\ 该工具可以对整个组织或选定区域自动分割细胞，并可通过调整参数优化结果。当初始分割结果较好但需进一步调优，或细胞密度变化较大时，建议使用该方法。
  3. 手动精修工具\ 使用套索、画笔和橡皮擦等工具手动对分割区域进行细致调整。适用于小区域内的精确修正，例如分离聚集细胞、修正分割错误或处理自动方法难以应对的复杂区域。更适用于小型数据集或需要逐个细胞精细调整的场景。

导入细胞分割掩膜

如果你已经使用第三方分割工具创建了 .tif 格式二值化掩膜文件，可通过以下方式导入：

  • 在右侧面板点击 Segmentation mask下拉菜单
  • 点击 Add mask 来导入掩膜文件
  • 选择本地的 .tif 掩膜文件后点击 Open 打开，即可导入

如果对导入后的结果不满意，还可以替换掩膜：

  1. 再次打开 Segmentation mask下拉菜单
  2. 在 CUSTOM 分类中，选择已有掩膜后点击
  3. 选择新文件并点击 Open 完成替换

参数可调的半自动工具

参数可调的半自动工具在全自动和全手动之间提供了平衡，允许您通过调整参数来优化细胞分割的效果，确保可以更好的适配您的组织样本。

  1. 点击右侧面板中的 Automated Segmentation 下拉菜单，选择 Watershed， Box select 鼠标工具会被激活。

  2. 使用 Box select 鼠标工具选择您需要调整的感兴趣区域（ROI）。界面会弹出 Watershed 参数设置面板。

  3. 在弹出的分水岭参数面板中，您可以调整算法参数优化您的 ROI 分割效果，可以参考分割算法参数说明和优化细胞分割的结果建议提示获取更多效果。

  4. 调整参数后，点击 Apply 系统会开始处理分割结果。请等待分割结果并确认是否符合预期。

  5. 如果大多数细胞的分割效果不错，但还有少数不准确的地方，建议使用手动精修工具（套索、画笔或橡皮擦）进一步修正。若无需进一步修改，可直接点击 Next 进入最后的步骤。

    注意，自动工具只会更新**已选定区域的分割结果**，因此选区边缘的细胞可能会出现边界比较**生硬或者半个细胞**的问题。

为避免此问题，可以选择整个组织区域，以确保边界区域的平滑分割，或者您也可以使用**手动精修工具**修正边缘或者其它误分类的细胞，同时也保持其余部分分割的准确性。

手动精修工具

这种方法可以提供最高的精度，通过使用套索、画笔和橡皮擦等工具手动调整分割结果，细化每个细胞的边界。

  • 套索工具：适合取消选择较大的区域，如背景。
  • 画笔和橡皮擦工具：适用于细化较小的区域，例如标记单个细胞或分离细胞群。

第五步：导出

在最后一步，您需要导出图像配准、组织区域划分以及细胞分割的全部处理结果。点击右侧的 Export image processing record，系统将生成一个 .tar.gz 文件，包含所有记录，方便后续分析和保存。

单击导出将打开您的资源管理器，选择保存的文件路径。

导出的结果包括两种类型的文件：

  • *.tar.gz 文件：保存了您上传的原始图像数据和手动处理的记录。它是 SAW 分析中将图像与表达矩阵数据结合的关键，文件结构是固定的，因此不建议手动更改内容，以避免数据读取错误。
  • *regist.tif 文件：如果您对图像配准进行了手动调整，系统会自动将调整后的图像保存为 TIFF 格式。是经过裁剪和调整的配准后的图像，跟特征表达矩阵尺寸相同，便于在第三方工具中使用，也支持回传至 StereoMap 继续分析。如果您未做手动配准，该 TIFF 文件可以在 SAW 的输出目录 /outs/文件夹中找到。

手动后的图像结果 TAR.GZ 接回 SAW

将手动后的图像处理结果传入 SAW 有两种方式：

  1. 使用 SAW count 流程的 --image-tar 参数

    • 运行SAW count将手动后的图像文件.tar.gz作为--image-tar的入参。
    • SAW count流程将 Stereo-seq 测序的 FASTQ 文件与手动后的图像文件一起分析。

    • 在分析完成后，系统会生成一个包含图像与测序结果整合的 HTML 报告，方便您查看和下载。

      cd /saw/runs

saw count \
    --id=<ID> \
    --sn=<SN> \
    --omics=<OMICS> \
    --kit-version=<TEXT> \
    --sequencing-type=<TEXT> \
    --chip-mask=/path/to/chip/mask \
    --organism=<organism> \
    --tissue=<tissue> \
    --fastqs=/path/to/fastq/folders \
    --reference=/path/to/reference/folder \
    --image-tar=/path/to/image/tar

  2. 使用SAW realign流程的--realigned-image-tar 参数

    • 运行SAW realign将，手动后的图像文件.tar.gz作为--realigned-image-tar的入参。
    • SAW realign流程跳过比对步骤。
    • SAW realign流程基于手动后的结果文件重新生成配准后的图像数据，并提取组织与单细胞层级的表达矩阵。

    • 流程结束后会生成一份新的 HTML 报告，包含更精准的细胞分割与空间表达矩阵的分析结果。

      cd /saw/runs

saw realign \
    --id=<ID> \
    --sn=<SN> \
    --count-data=/path/to/previous/SAW/count/task/folder/id \
    --realigned-image-tar=/path/to/realigned/image/tar
