参数可调的半自动工具

什么是可调参数的半自动工具

可调参数的半自动工具是一种将用户自定义参数调整与自动化算法相结合的图像分割工具。它允许用户通过微调关键参数（例如阈值、细胞大小和荧光强度）来灵活控制细胞分割的结果，同时利用算法进行高效的批量处理。这种工具介于完全手动（精度高但耗时久）与完全自动化方法（如深度学习，依赖大量数据集且可能缺乏适应性）之间。特别适用于以下场景：

  • 中等规模数据处理：比手动标注更高效，同时避免了完全自动化工具在训练数据不全面时可能产生的错误。
  • 多样化实验条件：灵活的参数调整适应不同的染色类型（如DAPI、H&E和多重免疫荧光）和变化的细胞密度（比如稀疏区与密集区）。
  • 可重复性：一旦设定参数，可以重复使用这些参数，确保一致且可重复的分割结果。

这种方法兼顾了速度、灵活性和精确度，成为生物学图像分析中的强大工具，广泛应用于科研和临床领域。

分割算法参数说明

细胞核参数

参数说明示例
Background radius浮点数（≥0，默认值为8.00）。背景估计的半径，建议设置的值要大于最大的细胞核半径，或设置为0关闭背景估计。

提示：对于细胞密集区域，建议设置较大的半径值以获得更平滑的背景校正效果。
Use opening by reconstruction

布尔值（默认开启）。重建开运算往往会给出“更好”的背景估计，因为它在用于细胞检测的图像块中包含了更多信息。然而，在某些情况下（例如，具有明显褶皱、背景染色或其他伪影的图像），这可能会导致问题，因为背景估计在各个图块之间会有很大差异。


提示：如果出现奇怪的背景效果，可以尝试关闭这个选项看看。

Median filter radius

浮点数（≥0，默认为0.00）。中值滤波器的半径，在处理每个像素时所考虑邻域像素的范围，消除孤立的噪点


提示：对于低对比度图像，建议适当增大中值滤波器的半径，可以覆盖更大的邻域像素范围，从而更有效地去除噪声，但是可能会导致图像模糊，影响分割的精度。

Sigma浮点数（≥0，默认为1.50）。对图像进行高斯模糊处理的半径大小，可以平滑图像减少噪声对背景估计的干扰，帮助算法更好的识别细胞边界。。

提示：对于噪声较多的图像，增加模糊半径可以减少噪声，防止细胞核碎片化，提升平滑噪声的效果，但可能会导致细胞边界不清晰。对于细节丰富的图像，减小模糊半径以保留细胞边界信息。
Minimum area浮点数（≥0，默认为10.00）。细胞核的最小面积阈值，如果检测到的细胞和面积小于设定的值，该细胞将会被丢弃。

提示：根据细胞的实际大小设置最小面积阈值。对于小细胞（如淋巴细胞），设置较小的最小面积
Maximum area浮点数（≥0，默认为400.00）。细胞核的最大面积阈值，如果检测到的细胞和面积大于设定的值，该细胞将会被丢弃。

提示：根据细胞的实际大小设置最大面积阈值。对于大细胞（如肿瘤细胞），设置较大的最大面积。

图像强度参数

参数说明示例
Threshold

浮点数（≥0）。强度阈值，区分细胞和背景的灰度值界限，检测到的细胞核像素值需要大于该阈值，否则就会被当做背景丢弃。

默认值：

  • 8-bit 灰度图像：25.00
  • 16-bit 灰度图像：100.00
  • 彩色图像：0.1


提示：请根据图像的类型和背景复杂度设置该值。

Smooth boundaries

布尔值（默认开启）。平滑细胞或者细胞核的边界。


提示：如果不勾选，边界将会有很多的锯齿，建议开启，以获得更清晰的分割掩膜。

优化细胞分割结果的建议

微调参数的推荐顺序

建议选择较小的区域微调参数，这样可以快速计算并查看结果。

注意，自动工具只会更新已选定区域的分割结果，因此选区边缘的细胞可能会出现边界比较生硬或者半个细胞的问题。

为避免此问题，可以选择整个组织区域，以确保边界区域的平滑分割，或者您也可以使用手动精修工具修正边缘或者其它误分类的细胞，同时也保持其余部分分割的准确性。

  1. Minimum & Maximum Area 确定细胞形态
    • 调整细胞大小范围： 以排除明显不符合细胞大小的区域或错误检测。

  2. Median Filter Radius

    • 增大半径： 去除纹理噪声或小的伪影，但过大的值可能导致细胞结构丧失。

  1. Sigma (Gaussian Smoothing)

    • 增大半径： 降低噪声，平滑边界。
    • 减小半径： 锐化细胞边缘，并保留更多细节，例如细胞间的边界。

    • 稀疏细胞区域： 使用较大的半径来平滑背景修正。
    • 密集细胞区域： 使用较小的半径以保留更多细节。

不同的染色类型

一般来说，Watershed 分割算法在灰度图像（如ssDNA、DAPI、IF）上的表现优于彩色图像（如H&E）。这是因为该算法将图像视作一个地形表面，其中高强度值（如细胞边界））表示山峰，低强度值（（如细胞内部、背景））代表山谷，其核心是基于图像的 “梯度”（明暗变化剧烈程度）来划分细胞核和背景，。对于灰度图像来说，这种关系非常直接，使得分割效果更加有效. (docs.opencv.org)

与此不同，彩色图像因为有多个通道（如红、绿、蓝），这可能会使分割更加复杂。为简化处理过程，彩色图像通常会在应用 Watershed 算法之前转换为灰度图像，并将灰度值做 0~1 的归一化。

灰度图像

  • Threshold: 设置较高的强度阈值，以有效地将细胞核与背景分离。
  • Minimum/Maximum Area: 根据预期的细胞大小进行调整，以过滤噪声和伪影。

彩色图像

  • Sigma: 增加该值以减少噪声并平滑图像边界。
  • Threshold: 设置小于等于1 的强度阈值，更好的分割结果。

密集与稀疏细胞区域

对于密集细胞区域：

  • Background Radius: 使用较小的背景半径以保留细节。
  • Maximum Area: 降低该值以防止细胞合并。

对于稀疏细胞区域：

  • Background Radius: 使用较大的背景半径，以更好地去除大范围的背景变化。
  • Minimum Area: 增加细胞大小，以减少背景噪声的误识别。

问题排查及常见问题

Q: 为什么不同类型图像的 Threshold（阈值）默认设置不同?

A: 在自动分割（如 Watershed）过程中，像素强度的阈值会受到图像类型和背景复杂度的影响。为了适应不同图像的动态范围和信噪比，系统为不同类型的图像设定了不同的默认阈值：

  • 8-bit 灰度图像：25.00（范围 0 ~ 255）
  • 16-bit 灰度图像：100.00（范围 0 ~ 65,535）
  • 彩色图像：0.1（范围 0 ~ 1）

这些默认值为了在大多数常见情况下提供良好的前景（如细胞）与背景的分离效果。当然您也可以根据图像的具体情况进行手动调整。
