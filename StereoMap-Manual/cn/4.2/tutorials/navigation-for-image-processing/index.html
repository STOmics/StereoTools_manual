图像处理指南

为什么在空间分析中要使用图像数据

每一种组织在临床上的特征表达（如 RNA 和蛋白质）可能不均匀，仅基于空间的表达信息去准确识别组织/细胞的边界将会具有较大的挑战。细胞核染色（比如 ssDNA 荧光染色）或者苏木精-伊红（ H&E 染色）染色的组织图像在显微镜下可以清晰的显示整个组织区域，使用染色的图像可以显著改善组织或者细胞的轮廓信息。在确定组织边界后，可以将图像数据与空间表达分析的数据对齐，并利用边界信息获取组织或者细胞区域的热图，以进行更深层次的分析。

Stereo-seq 芯片表面一些规则排列的横线和竖线（称为轨迹线），用于帮助识别碱基和配准图像。这些线是探针没有覆盖的区域，在空间表达的热图中会显示为狭窄的细线。

为了获得更好的图像配准效果，Stereo-seq 的组织染色和成像流程经过专门优化，不仅提高了显微镜下轨迹线的的可见性，同时也不会影响下游 mRNA 捕获的效率。

由于轨迹线会同时出现在表达热图和显微镜图像中，它们可以作为图像和空间数据配准的精准参考点。

Stereo-seq 分析流程软件包（SAW）嵌入了自动图像处理算法，能够准确识别组织和细胞的边界，并检测 Stereo-seq 芯片上轨迹线。轨迹线是将显微镜图像与空间表达矩阵进行配准的重要参考。如果无法检测到轨迹线，或者组织和细胞边界不清晰，您可能需要手动勾画边界或手动对齐图像，以保证配准的准确性。

图像处理用户路线图

推荐的图像处理用户路线图如下：

  1. 检查显微镜图像的质量。这是Stereo-seq 实验流程中的重要环节，其目的是为了确认轨迹线是否清晰可见，图像拼接是否准确以及组织结构是否能够被容易识别。我们建议在实验阶段就使用图像 QC 工具进行质控，以便后续分析更顺利。若未及时质控，也可在图像处理模块中再次进行检查。具体的评估标准可参考 图像质控 Image QC 获取更多详情。
  2. 图像与表达矩阵热图的配准。将图像与表达矩阵的热图重叠，确保图像的方向、尺度和角度等跟表达矩阵一致。如有多张免疫荧光（IF）的图像，请分别确认它们的配准结果。如您完成配准后，可以导出手动处理的结果（.tar.gz）和 TIFF 格式的配准后图像（.tif），使用第三方工具或算法进一步处理（如细胞分割）。
  3. 圈选组织和细胞。您可以使用交互式绘图工具、半自动识别工具（支持调节参数），或导入外部生成的掩膜文件。对于免疫荧光 IF 图像，可以通过调节阈值区间获得 ROI 区域，因亮度较低的区域通常是背景噪声，可以通过设置强度阈值来更准确地选定 ROI。精确的区域选择对于确保高质量数据提取、减少背景干扰至关重要。
  4. 导出操作记录文件。手动后的记录可以经过 SAW 的realgn流程提取组织或者细胞区域对应的空间表达矩阵。
