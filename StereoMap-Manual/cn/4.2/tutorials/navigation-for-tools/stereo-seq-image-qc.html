图像质控-Image QC

图像质控 (Image QC) 用于评估 Stereo-seq 实验拍摄的显微镜图像是否可以在 Stereo-seq 分析流程软件包(SAW) 中准确的自动分析。

QC 入口

从 StereoMap 启动页的 Tools 进入 Image QC。

输入图像建议

本节介绍可用于图像质控（Image QC）的图像类型、推荐的文件格式，以及与不同显微镜系统的兼容性。

图像类型

StereoMap 图像质控模块支持以下类型的图像输入：

  • 细胞核染色图像（如 ssDNA 或 DAPI）
  • 免疫荧光图像：包含一张细胞核染色图像（如 DAPI）和最多 6 张免疫荧光（IF）图像
  • 彩色图像：特指 H&E 染色图像

QC 指标及标准

图像的质量直接反映了显微镜的稳定性和成像系统的优劣。在图像质控（Image QC）中，图像基本的评估包括 track 线评估，当然可能会根据染色类型的不同，增加校准评估。具体的评估标准和评分阈值会根据图像类型略有差异。

图像质控评估的指标包括：

Track 线评估

  • Track 线是刻在 Stereo-seq 芯片表面的标记线。 显微镜拍摄的图像和空间特征表达矩阵都可以被检测到 track 线，可以将这两种数据模态结合起来，用于将图像跟表达矩阵的对齐。Track 线得分综合考虑了 track 线的清晰度（是否可见）及相邻 track 线的数量等因素，算法根据检测的 track 线，可以计算图像的角度、尺度和位置等并推导出图像的全局模板，以评估图像是否可以自动成功配准。

校准评估

  • 在一个芯片的多组图像数据中，每组图像的拼接、配准和组织区域范围必须保持严格的一致性才可以保证跟对应的表达矩阵对齐。由于染色条件和成像系统的限制，跟矩阵配准的图像仅在 DAPI 核染色图像中可见，因此必须保证其它 IF 图像首先跟 DAPI 图像对齐，然后使用相同的配准参数将各个图像与空间特征表达矩阵做配准，算法使用图像的相似性和特征偏移评估两张图像是否对齐。该评估指标仅适用于 DAPI+mIF 的单通道图像场景。
评估指标分数示例
Track线评估
  • ≥ 60，算法检测到足够数量的合个的 track 线，图像可以与空间表达矩阵做自动配准。
  • < 60，算法检测到较少的 track 线或者未检测出 track线，无法推导出全局模板，您可能需要重新拍照。

QC 是否通过的必要条件。

校准评估
  • 通过，核染色图像（DAPI）与另一图像(IF)之间的最大偏移量 ≤ 20 像素，且图像的形态相似度 ≥ 1%。
  • 未通过，核染色图像（DAPI）与另一图像(IF)之间的最大偏移量 > 20 像素，或者图像的形态相似度 < 1%。

有条件的评估指标，只适用于mIF。校准评估的通过与否不影响整体质控（QC）结果的判定。

备注：可以参考 Stereo-seq 空间多组学技术成像指南的 4.3 章节图片示例获取更多图像相关的信息。

图像 QC 操作指南

下面展示了图像 QC 的操作界面：

开始 QC

  • 打开图像 QC 小工具，拖入待QC的文件。有关图像文件命名的更多实用建议，请参考针对不同图像的 QC 建议，避免处理错误。
  • 部分图像信息会自动填写，确保至少填写必填字段并保证其准确性。
图像信息描述及例子
Chip SN【必选】 Stereo-seq 芯片序列号，可以在 Stereo-seq 芯片底部找到。

E.g. S1 (1x1) 芯片: SS200000135TL_D1, C02533C
S0.5 (0.5x0.5) 芯片: FP200009107_E414, B03210C211
大芯片: S200000108BR_A3A4, D02070C3D3
请确保填写的 SN 跟实际拍摄图像的 SN 是对应的。
Operator【必选】 用户信息，建议输入拍摄人员的邮箱地址。
Image Path【必选】 待 QC 的文件目录，拖入后自动填充。

建议拖入的文件路径中不能包含字母、数字和下划线以外的任何字符。
Staining Types【必选】 图像的染色类型，包括 ssDNA，DAPI，DAPI+mIF和H&E。
Upload

【必选】 将图像数据上传到 STOmcis Cloud 或者用户自定义的路径，选项包括：

  • No：不上传（默认）。
  • QC Input Files ：仅上传待 QC 的原始文件。
  • QC Output Files (TAR.GZ): 仅上传 QC 后的文件。
  • Select all: 上传待 QC 的原始文件及 QC 后的文件.

可以查阅上传设置获取更多信息。

Remark【可选】 输入图像或者 QC 的额外描述信息，用于记录各类不同的情况。

运行 QC

当您填写完所有必需的字段后，运行按钮会变为可点击状态，您就可以开始 QC 评估了。

图像质控全流程可以分为几个步骤：

  1. QC指数评估：我们会对图像的 track 线和校准做评估。
  2. 图像压缩：系统会生成压缩后的 QC 结果文件，供后续使用。
  3. （可选）QC文件上传：如果启用了此功能，图像将在评估过程中上传。

在整个评估的过程中，您可以实时查看图像质控指标的进度、结果和 QC 的最终结论。如果需要，系统会根据评估结果提供一些进一步的分析建议。

多个图像同时QC

如果您需要对多个图像同时做 QC，您可以为每张图片添加单独的 tab 页面，填写必要的信息后提交任务，所有的任务将在后台按顺序逐一执行。

查看 QC 的结果文件

QC 运行完成后，可以直接在界面中查看QC 后的结果文件路径。您也可以在 StereoMapWorkspace -> QC 保存路径中查看 QC 后的结果文件，保存路径可以在设置中更改。

QC 完成的后续步骤

如果您的图像数据是 QC 成功的。

您可以使用--image-tar参数，将 QC 输出的.tar.gz传入到SAW count流程，这样系统就可以自动将图像和特征表达矩阵一起处理。

如果您的图像数据是 QC 失败的。

如果您的图像数据 QC 未通过，您仍然可以将图像跟矩阵在可视化共同展示。只需要将图像文件输入StereoMap 的Image Processing 做手动处理后再接回 SAW即可。

针对不同图像的 QC 建议

不同类型的图像在图像 QC 的过程中有不同的注意事项。为保证分析准确性，我们为 ssDNA/DAPI、DAPI+mIF 和 H&E 图像分别列出了一些实用的小贴士。

核染色图像：ssDNA 或者 DAPI

核染色图像的评估主要基于：

  • track 线评估（必要的质控指标）

H&E图像：H&E

H&E 图像的评估主要基于：

  • track 线评估（必要的质控指标）

确保在显微镜下可以看到清晰的 track 线是至关重要的，对于算法来说，识别 H&E 彩色图像的 track 线比灰度图像具有更大的挑战。

  • 以下几种情况可能会让时间变慢：
    • 彩色图像（因 H&E 图像是一种彩色的图像，其大小约为灰度图像的三倍）

自定义的上传路径

配置自定义上传路径的步骤

  1. 选择云服务类型

    • 选择您购买的云服务类型 ALICLOUD.CUSTOM 或 AWS.CUSTOM。
    • 区域选择 **OWNER
